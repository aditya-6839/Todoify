## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- MongoDB (local or Atlas account)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/todoify.git
   cd todoify
   ```

2. **Install dependencies**

   Frontend:
   ```bash
   cd client
   npm install
   ```

   Backend:
   ```bash
   cd server
   npm install
   ```

3. **Environment Setup**

   Create a `.env` file in the `server` directory:
   ```env
   NODE_ENV=development
   PORT=5000
   
   # Database
   MONGO_URI=your_mongodb_connection_string
   
   # JWT
   JWT_SECRET=your_jwt_secret_key
   JWT_EXPIRE=7d
   
   # Frontend URL (for CORS)
   CLIENT_URL=http://localhost:5173
   ```

   Create a `.env` file in the `client` directory:
   ```env
   VITE_API_URL=http://localhost:5000/api/v1
   ```

4. **Run the application**

   Backend (from `server` directory):
   ```bash
   npm run dev
   ```

   Frontend (from `client` directory):
   ```bash
   npm run dev
   ```

5. **Access the app**
   - Frontend: `http://localhost:5173`
   - Backend API: `http://localhost:5000/api/v1`

## ğŸ“¡ API Documentation

### Standard Response Format

```json
{
  "success": true,
  "message": "Operation successful",
  "data": {}
}
```

### Authentication Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/v1/auth/register` | Register new user |
| POST | `/api/v1/auth/login` | Login user |
| GET | `/api/v1/auth/me` | Get current user |
| POST | `/api/v1/auth/logout` | Logout user |

### Task Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/v1/tasks` | Create new task |
| GET | `/api/v1/tasks?view=inbox\|today\|upcoming` | Get tasks with filters |
| GET | `/api/v1/tasks/:id` | Get single task |
| PUT | `/api/v1/tasks/:id` | Update task |
| DELETE | `/api/v1/tasks/:id` | Delete task |
| PATCH | `/api/v1/tasks/:id/toggle` | Toggle task completion |

### Project Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/v1/projects` | Create project |
| GET | `/api/v1/projects` | Get all user projects |
| PUT | `/api/v1/projects/:id` | Update project |
| DELETE | `/api/v1/projects/:id` | Delete project |

### Label Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/v1/labels` | Create label |
| GET | `/api/v1/labels` | Get all user labels |
| PUT | `/api/v1/labels/:id` | Update label |
| DELETE | `/api/v1/labels/:id` | Delete label |
